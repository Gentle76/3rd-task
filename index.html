What does 'a href' do in an html document?

The 'a' tag in HTML is used to create hyperlinks, allowing users to navigate to different web pages or resources. it stands for "anchor" and is commonly known as the anchor tag

What distinguishs normal HTML elements from semantic HTML elements?

Normal HTML elements primarily serve for structural purposes, while Semantic HTML elements provide additional meaning and context to the content

What is the difference between closing tags and self-closing tags in in Html?

In HTML, closing tags are used to mark the end of an element's content, while self-closing tags are used for elements that don't have any content or don't require closing tags.

What distinguishes inline elements from block elements in HTML

Inline elements are those that don't start on a new line and only take up as much width as necessary, such as "span" or "a". Block elements, on the other hand, start on a new line and take up the full width available, such as "div" or "p". Block elements can contain other block and inline elements, while inline elements are typically contained within block elements.
